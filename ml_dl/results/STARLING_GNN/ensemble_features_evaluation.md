# 系综物理化学特征评估报告

## 1. 特征列表与含义

| 特征 | 含义 | 单位 |
|------|------|------|
| N | 残基数（链长） | — |
| Rg_mean / Rg_std | 回转半径的系综均值与标准差 | Å |
| REE_mean / REE_std | 端到端距离的系综均值与标准差 | Å |
| asphericity_mean / asphericity_std | 形状参数（0≈球、1≈棒）的系综均值与标准差 | — |
| entropy_nats / entropy_norm | 构象熵（基于 Rg/REE/asphericity 分箱） | nats / 归一化 [0,1] |
| RMSD_centroid_mean / RMSD_centroid_std | 各构象相对系综质心结构的 RMSD 的均值与标准差 | Å |

---

## 2. 区分能力（idr_pos vs neg）

- **样本量**：idr_pos n=147，neg n=458。
- **Mann-Whitney U 检验**：所有特征的 p 值均 > 0.29，**无特征在 0.05 水平上显著区分**两类。
- **效应量（Cohen's d）**：全部在 ±0.2 以内，属 **negligible**；最大绝对值为 asphericity_mean（d ≈ -0.17）。

**结论**：当前这批**标量系综特征**单独或线性组合下，**不足以区分**能相分离（idr_pos）与不能相分离（neg）。与几何 GNN（AUC≈0.80）相比，说明有用信息更可能来自**高维/局部/拓扑结构**，而非这些全局标量汇总。

---

## 3. 特征间相关性

- **链长 N** 与 Rg_mean、REE_mean、RMSD_centroid_mean 等高度相关（r ≈ 0.89–0.92），即**尺寸类指标主要由链长驱动**。
- **Rg_mean、REE_mean、RMSD_centroid_mean** 两两高度相关（r ≈ 0.97–0.99），反映同一物理量（链的“伸展/尺寸”）的不同度量，**信息重叠大**。
- **entropy_nats** 与 **entropy_norm** 完全共线性（r=1.0），保留其一即可。
- **Rg_std、REE_std、RMSD_centroid_std** 彼此也高度相关（r ≈ 0.89–0.99），均为“系综内波动”的度量。

**结论**：若用于回归/分类，存在**强多重共线性**；建议做特征筛选或降维（如只保留 N + Rg_mean + asphericity_mean + entropy_norm + RMSD_centroid_mean），或使用正则化/树模型。

---

## 4. 质量与合理性

- **数值范围**：Rg、REE、RMSD 约 2–6 Å、6–15 Å、1.5–2.5 Å 量级，与 Cα 系综尺度一致；asphericity 多在 0.5–0.6，偏棒状；entropy_norm 约 0.89，说明 40 帧在 (Rg, REE, asphericity) 空间分布较分散，**计算合理**。
- **缺失**：605 个蛋白全部成功计算，无缺失值。
- **熵的定义**：当前为基于 5×5×5 分箱的离散熵，对 40 帧较粗糙；若需更细粒度可增加分箱数或改用聚类熵，但趋势应与当前一致。

---

## 5. 总结与建议

| 维度 | 评估 |
|------|------|
| **区分 idr_pos / neg** | 差：无显著特征，效应量可忽略。 |
| **信息冗余** | 高：N/Rg/REE/RMSD 强相关；entropy_nats 与 entropy_norm 重复。 |
| **计算与量纲** | 良好：无缺失，量纲与文献常用一致。 |

**建议**：
1. **解释/报告**：这些特征适合用于描述系综整体几何与构象多样性，**不宜**作为“相分离与否”的单一判据；与 GNN 结合时，可作为辅助输入或对照基线。
2. **建模**：若用这些特征做传统 ML，建议先筛或降维（如 PCA），并优先用树模型或正则化线性模型以应对共线性。
3. **扩展**：若希望提升区分度，可考虑**序列/局部**特征（如残基类型、短程接触、带电/疏水块）或**更高维构象描述**（如接触图分布、序参数），而非仅依赖当前全局标量。
